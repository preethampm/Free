Votu is a live event feedback platform. Create an event, place QR codes at each booth or stall, let attendees scan and rate on the spot, and watch the leaderboard update in real time — no app download required.

## How it works

**For organizers**

1. Sign in with Google
2. Create an event (e.g. "College Fest 2026")
3. Add items for people to rate (games, stalls, presentations, etc.)
4. Set rating criteria for each item (fun factor, presentation, creativity, etc.)
5. Download QR codes — one per item
6. Print and place QR codes at each location
7. Watch results come in live on the leaderboard

**For attendees**

1. Scan the QR code at any item
2. Enter your name and phone number
3. Rate each criterion
4. Leave optional written feedback
5. Rate all items for your votes to count

## What you get

- QR codes per item — print and stick at the booth
- Real-time leaderboard — scores update as votes come in
- Per-criterion breakdown — not just one average score
- Progress tracking — attendees see how many items they've rated
- CSV export — download full results anytime
